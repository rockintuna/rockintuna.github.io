07. 문자 자료형

인코딩 : 문자 -> 숫자값(코드)
‘A’ -> 65
디코딩 : 코드 -> 문자
65 -> ‘A’

문자세트 (code-set)

  1. 아스키(ASCII) : 1byte 영문자, 숫자, 특수문자 표현
  2. 유니코드(Unicode) : 한글 등 복잡한 언어 표현
    ‘가’ -> AC00 (2byte)

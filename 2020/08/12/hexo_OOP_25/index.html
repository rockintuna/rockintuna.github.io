25. 상속이란

클래스에서 상속의 의미 :
새로운 클래스를 정의할 때 이미 구현된 클래스를 상속받아서
속성이나 기능이 확장되는 클래스를 구현 함.

상속하는 클래스 : 상위 클래스
상속받는 클래스 : 하위 클래스

자바에서는 다중 상속이 불가능하다.

상속을 사용하는 경우 :
상위클래스는 더 일반적인 개념과 기능을 가짐
하위클래스는 더 구체적인 개념과 기능을 가짐

10. 대입, 부호, 산술, 복합대입, 증감 연산자

항 : 연산에 사용되는 값
연산자 : 항을 이용하여 연산하는 기호

  • 대입 연산자
    int age = 20;
    우선순위 가장 낮음.

  • 단항 연산자
    부호 유지 또는 변경 (+,-)

    + 부호 유지
    - 부호 변경
    실제 변수 값의 부호를 바꾸려면 대입 연산자와 같이 사용.

  • 산술 연산자
    사칙연산, %(나머지)

  • 복합 대입 연산자
    대입 연산자 앞의 연산자의 결과를 l-value에 대입

  • 증가 감소 연산자
    변수의 값을 1 더하거나 뺄때 사용

증감 연산자는 대입을 내포한다.
